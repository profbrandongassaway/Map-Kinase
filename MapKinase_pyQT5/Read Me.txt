M1_hsa_annotations.py
    Function
        Adds Kegg annotations (like hsa) to our proteomic dataset
    Input
        Protein Dataset (tab delimited .txt), Header of Uniprot ID column (str)
    Output
        Output file name and location (str)
    Options

M2_protkegg_plot_matplotlib.py
    Function
        Plots our protein dataset onto a Kegg Pathway.
    Input
        Protein Dataset (tab delimited .txt), kegg pathway id (str), Header of fold change column (str),
        Header of hsa column (str), Header of gene symbol column (str)
    Output
        Output file name and location (str)
    Options
        Positive Color (str) ex: (0, 0, 1), Negative Color (str) ex: (1, 0, 0),
        Max positive fold change value (int or float) ex: 2.5, Max negative fold change value (int or float) ex: -2.5,
        Protein selection option (1 for first in list, 2 for most significant fold change)

M3_transfer_kegg_annotations.py
    Function
        Takes the Kegg annotations from an input protein dataset (went through M1 at some point), and uses
        the Uniprot ID column to copy them unto the corresponding input phosphosite dataset.
    Input
        Protein Dataset (tab delimited .txt), Header of Uniprot ID column in Protein Dataset (str),
        Header of kegg annotation column in Protein Dataset (str),
        Phosphosite Dataset (tab delimited .txt), Header of Uniprot ID column in Phosphosite Dataset (str),
    Output
        Output file name and location (str)
    Options

M4_phoskegg_plot_matplotlib.py
    Function
        Plots our protein and phosphosite dataset onto a Kegg Pathway.
    Input
        kegg pathway id (str),
        Protein Dataset (tab delimited .txt), Header of fold change column in Protein dataset (str),
        Phosphosite Dataset (tab delimited .txt), Header of fold change column in Phosphosite dataset (str),
        Header of hsa column in Protein dataset (str), Header of gene symbol column in protein dataset (str),
        Header of Modulation PSP column in phosphosite dataset (str),
        Header of phosphosite column in phosphosite dataset (str),
    Output
        Output file name and location (str)
    Options
        Positive Color (str) ex: (0, 0, 1), Negative Color (str) ex: (1, 0, 0),
        Max positive fold change value (int or float) ex: 2.5, Max negative fold change value (int or float) ex: -2.5,
        Protein selection option (1 for first in list, 2 for most significant fold change),
        Phosphosite selection option (1: Largest fold change, 2: Modulating sites prioritized, 3: Only modulating sites),
        Radius of phosphorylation circles (int) ex: 5, Space between phosphorylation circles (int) ex: 2,
        Maximum number of phosphorylation sites to display per protein (int) ex: 4,
        Layout of the phosphorylation sites ('linear' or 'parameter'),


Summary of Data Flow
m1 → m7: DataSelectionUI sends selected_data to CombinedUI.update_selected_data.
m7 → m5: CombinedUI passes self.selected_data to VariableSetterUI.update_data.
m5: VariableSetterUI stores data in self.input_data and uses it in run_visualization.
m7 Load: load_all_data loads a JSON file and calls VariableSetterUI.set_current_data, which sets self.input_data.
